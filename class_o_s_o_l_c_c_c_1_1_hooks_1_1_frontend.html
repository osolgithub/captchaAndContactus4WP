Detailed Description

OSOL_CCC_Handler class: Used for OOPfying the frontend operations of wp plugin.


This class encapsulates all custom methods for for frontend of the plugin.
This class also encapsulates all frontend hooks for the plugin.

  • This class deals with

    1. display captcha
    2. generate captcha html for wordpress forms (login, register, lost password, post, comment)
    1. validate captcha on submission
    2. display contact us form
    3. process contact us form submission instantiation \OSOLCCC\Hooks\Frontend::getInstance()
