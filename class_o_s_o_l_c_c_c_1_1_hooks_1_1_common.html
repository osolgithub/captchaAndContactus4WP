Detailed Description

OSOL_CCC_Handler class: Used for OOPfying the wordpress plugin.


This class encapsulates all custom methods for for backend of the plugin.
This class also encapsulates all backend hooks for the plugin.

  • This class deals with
    1. Adding Plugin Configuration link in admin side
    2. display configuration options of captcha
    1. display "contact us" form configuration options
    2. Save/Update configuration forms submissions
