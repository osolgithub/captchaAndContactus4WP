== Installation ==

1.Download the plugin
2. From WordPress 4 , Admin Panel >> Plugins, Seach for "Customizable Captcha and Contact us". Click "install" & the "activate" . For older versions, Upload plugin to the `/wp-content/plugins/` directory. 
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Set the options for captcha and Contactus form in Settings Tab as in the screenshots 6 and 7
5. To use contact us form functionality,  insert the code [cccontact] in a page/post where you want to show contact us form.
