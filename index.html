Captcha and Contact Us Forms for Wordpress

Adds OSOLMulticaptcha (http://www.outsource-online.net/osolmulticaptcha-simplest-php-captcha-for-html-forms.html ) to wordpress forms for registration,login,forgot password and comment.

Additionally ,this will enable admin to add contact us form with the same captcha in any wordpress page just need to insert the code [cccontact] in a page where you want to show contact us form

Requirements
PHP GD Library must be available in the server safe mode must be turned off

The above requirements are default settings in most PHP hosts.however if the captcha isnt showing up you need to check those settings
